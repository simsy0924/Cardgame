# 불가사의 수동 테스트 체크리스트

- [ ] 불가사의한 밤의 대도시가 없으면 도시 의존 효과/소환 절차가 뜨지 않는다.
- [ ] 도시 의존 묘지/제외 소환은 `procedure`로 처리된다.
- [ ] `발동` 문구가 없는 불가사의 효과는 기본적으로 지속 효과로 분류된다.
- [ ] 처리 시 무효 효과는 별도 체인을 만들지 않는다.
- [ ] 처리 시 무효는 `negateTags`가 맞는 효과에만 적용된다.
- [ ] 태그 없는 효과는 처리 시 무효할 수 없다.
- [ ] 상대 선택이 필요한 효과는 임의로 내 화면에서 자동 선택하지 않는다.
