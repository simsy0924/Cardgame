# 서커스메어 수동 테스트 체크리스트

## 키 카드 소환 절차

- [ ] 서커스메어 키 카드 몬스터는 조건을 만족할 때만 소환 후보로 뜬다.
- [ ] `서커스메어 메드 키메라` 소재 후보는 메드 울프/베어/이글/씰 4종으로 제한된다.
- [ ] 마법/함정/필드 카드는 메드 키메라 소재 후보로 뜨지 않는다.
- [ ] `서커스메어 출입통제` 같은 비몬스터/범용 카드는 소재 후보로 뜨지 않는다.
- [ ] 범용 키카드는 키덱에는 들어가도 서커스메어 소환 후보에는 뜨지 않는다.

## 스프링 제스터 박스

- [ ] 소환 시 효과 후보가 trigger로 뜬다.
- [ ] 효과 해결 시 상대 필드 몬스터만 묘지로 보낸다.
- [ ] 자기 필드 몬스터는 건드리지 않는다.
- [ ] 상대 필드에 처리 가능한 몬스터가 없으면 발동 후보로 뜨지 않는다.
- [ ] 상대 몬스터가 모두 묘지 보내기 내성이면 발동 후보로 뜨지 않는다.

## 메드 키메라

- [ ] 지속 효과가 소환 유발 후보로 뜨지 않는다.
- [ ] 대상 내성이 실제 대상 지정 효과를 막는다.
- [ ] 상대 효과로 묘지로 보내기 내성이 적용된다.
- [ ] 필드에서 벗어나면 지속 효과가 사라진다.

## 악몽의 서커스장

- [ ] 패에서 발동하면 필드 존에 놓인다.
- [ ] 발동 무효 시 묘지로 간다.
- [ ] 필드 존에서 발동 가능한 효과 버튼이 뜬다.
- [ ] 지속 효과는 버튼으로 뜨지 않는다.
- [ ] 처리 불가능한 효과는 버튼으로 뜨지 않는다.
