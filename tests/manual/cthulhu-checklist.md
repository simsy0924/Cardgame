# 크툴루/올드원 수동 테스트 체크리스트

## 코스트/효과 분리

- [ ] `패에서 소환하고 발동` 구조에서 소환 선처리/코스트가 먼저 처리된다.
- [ ] 효과가 무효되어도 선처리/코스트는 되돌아가지 않는다.
- [ ] 코스트를 지불할 수 없으면 발동 자체가 불가능하다.
- [ ] 효과 처리 대상이 없으면 발동 자체가 불가능하다.

## 태평양 속 르뤼에

- [ ] 패에서 발동하면 필드 존에 놓인다.
- [ ] 효과로 필드 존에 놓이면 발동 시 효과를 처리하지 않는다.
- [ ] 필드 존에서 ①/② 효과 버튼이 뜬다.
- [ ] 처리 불가능한 필드 존 효과는 버튼으로 뜨지 않는다.
- [ ] 르뤼에 ③ 처리 시 무효는 별도 체인을 만들지 않는다.
- [ ] 르뤼에 ③은 자기 GOO 관련 태그 효과에만 적용된다.
- [ ] 태그 없는 효과는 르뤼에 ③으로 무효할 수 없다.

## 아우터 갓

- [ ] 아우터 갓 니알라토텝 소환은 `procedure`로 처리된다.
- [ ] 아우터 갓 소환 조건은 일반 발동 효과 후보와 섞이지 않는다.
- [ ] 조건 불충족 시 소환 버튼이 뜨지 않는다.
