# 엘리멘츠 수동 테스트 체크리스트

## 자체 소환

- [ ] 정령 자체 소환은 체인 블록을 만들지 않는다.
- [ ] 정령 자체 소환은 자신 전개 단계에서만 가능하다.
- [ ] 체인이 열려 있으면 자체 소환 버튼이 뜨지 않는다.
- [ ] 몬스터 존이 가득 차면 자체 소환 버튼이 뜨지 않는다.

## 카운터

- [ ] 카운터는 카드 객체의 `counters`에 저장된다.
- [ ] 카운터가 올라간 뒤 UI에 유지된다.
- [ ] 카운터가 양쪽 화면에 동기화된다.
- [ ] 카운터 조건을 만족하지 않으면 궁극신/궁극 창조신 소환 버튼이 뜨지 않는다.

## 궁극신 / 궁극 창조신

- [ ] 궁극신 소환 절차는 `procedure`로 처리된다.
- [ ] 궁극 창조신 소환 절차는 `procedure`로 처리된다.
- [ ] 소환 절차는 체인 없이 바로 소환된다.
- [ ] 조건 불충족 시 소환 버튼이 뜨지 않는다.

## 필드 카드

- [ ] 엘리멘츠 필드 카드는 패에서 발동 시 필드 존에 놓인다.
- [ ] 발동 시 효과 처리는 체인 해결 시 1회만 처리된다.
- [ ] 효과로 필드 존에 놓이면 발동 시 효과 처리를 하지 않는다.
