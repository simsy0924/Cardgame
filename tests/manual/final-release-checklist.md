# 최종 릴리즈 수동 체크리스트

20단계 이후 실제 플레이 전에 확인한다.

## 자동 테스트 선행

- [ ] `node tests/engine/run-engine-tests.js`가 통과한다.
- [ ] 브라우저 콘솔에 스크립트 로딩 오류가 없다.
- [ ] `index.html`이 삭제된 구 테마 파일을 요청하지 않는다.

## 로비 / 덱 / 시작

- [ ] 모든 테마 프리셋이 로비에서 선택 가능하다.
- [ ] 서커스메어 프리셋이 보인다.
- [ ] 크툴루/올드원 프리셋이 보인다.
- [ ] 라이온/타이거/라이거 프리셋이 보인다.
- [ ] 게임 시작 후 패/덱/키덱 수가 비정상적으로 깨지지 않는다.

## 공통 효과 엔진

- [ ] 같은 효과 버튼이 2개 뜨지 않는다.
- [ ] 처리할 수 없는 효과 버튼은 아예 뜨지 않는다.
- [ ] 발동형 효과는 체인에 올라간다.
- [ ] 소환 절차는 체인 없이 실행된다.
- [ ] 지속 효과는 버튼으로 뜨지 않는다.
- [ ] 처리 시 무효는 일반 체인 응답처럼 뜨지 않는다.
- [ ] 필드 존 카드는 필드 존에서 효과 버튼이 뜬다.

## 테마 대표 스모크 테스트

- [ ] 펭귄: `각성의 펭귄 군단` 반복 효과가 확인/취소로 동작하고, 처리 불가능하면 자동 종료한다.
- [ ] 서커스메어: `스프링 제스터 박스`가 상대 필드 몬스터만 묘지로 보낸다.
- [ ] 엘리멘츠: 자체 소환/궁극신 절차가 체인 없이 동작한다.
- [ ] 크툴루: 르뤼에 필드 존 효과와 처리 시 무효가 분리되어 동작한다.
- [ ] 지배자/지배룡: 버려짐과 패 이외에서 묘지로 보내짐이 구분된다.
- [ ] 불가사의: 도시 의존 소환 절차와 처리 시 무효가 분리된다.
- [ ] 마피아: 상대 선택이 필요한 효과를 내 화면에서 임의 선택하지 않는다.
- [ ] 라이온/타이거/라이거: 존 확장/축소가 UI와 소환 가능 체크에 반영된다.

## 네트워크

- [ ] 효과 해결 후 양쪽 필드 수가 같다.
- [ ] 한쪽에서만 몬스터가 2장으로 보이는 일이 없다.
- [ ] 상대 효과를 내 클라이언트가 다시 resolve하지 않는다.
- [ ] 필드/묘지/제외 상태가 양쪽에서 일치한다.
- [ ] 상대 선택 요청이 실제 상대에게 전달된다.

## AI전

- [ ] AI전에서 필드가 깜빡이며 되돌아가지 않는다.
- [ ] AI가 해결한 효과를 내 클라이언트가 중복 해결하지 않는다.
- [ ] 장기전에서 묘지/제외/필드 수가 꼬이지 않는다.
