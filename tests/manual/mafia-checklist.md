# 마피아 수동 테스트 체크리스트

- [ ] 상대 효과 변형 효과는 원래 체인 링크의 resolve를 스킵한다.
- [ ] 변형된 효과만 1회 처리된다.
- [ ] 상대가 고르는 선택지는 실제 상대 클라이언트에 선택 요청을 보낸다.
- [ ] 내 화면에서 상대 선택을 임의 자동 선택하지 않는다.
- [ ] 마피아의 도시 발동 횟수 제한이 턴마다 유지된다.
- [ ] 처리 불가능한 변형 선택지만 남으면 효과를 발동할 수 없다.
