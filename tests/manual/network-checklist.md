# HandBattle 네트워크 수동 테스트 체크리스트

## 준비

- [ ] 브라우저 2개 또는 기기 2개 준비
- [ ] 한쪽은 방 생성
- [ ] 다른 쪽은 방 코드로 접속
- [ ] 양쪽 닉네임 표시 확인
- [ ] 초기 덱/패/키덱 수 확인

## 기본 동기화

- [ ] 한쪽에서 몬스터 1장 소환
- [ ] 양쪽 화면에서 같은 플레이어 필드에 몬스터 1장만 보임
- [ ] 한쪽에서 묘지로 보내기 효과 사용
- [ ] 양쪽 화면에서 필드/묘지 수가 동일함
- [ ] 제외 효과 사용 후 양쪽 제외 수가 동일함
- [ ] 패 공개 카드가 양쪽에서 같은 공개 상태로 보임

## 체인 해결

- [ ] 발동자가 체인을 해결함
- [ ] 상대 클라이언트가 같은 효과를 다시 해결하지 않음
- [ ] 체인 해결 후 양쪽 로그 순서가 동일함
- [ ] 처리 시 무효가 적용될 때 원래 효과가 양쪽 모두에서 스킵됨
- [ ] 효과 변형이 적용될 때 원래 효과가 양쪽 모두에서 스킵됨

## 필드 카드 / 지속 효과

- [ ] 필드 카드를 발동한 쪽과 상대 쪽의 필드 존 카드가 동일함
- [ ] 필드 카드가 무효되면 양쪽 모두 묘지로 이동함
- [ ] 지속 효과로 몬스터 존 수가 바뀌면 양쪽 UI가 같은 슬롯 수를 보여줌

## 새로고침 / 복구

- [ ] 체인 해결 후 한쪽 새로고침
- [ ] 새로고침 후 필드/묘지/제외/필드 존 상태가 유지됨
- [ ] 새로고침 후 같은 효과가 다시 해결되지 않음
