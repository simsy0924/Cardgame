# 펭귄 수동 테스트 체크리스트

## 공개 패 / 대체 처리

- [ ] 펭귄 마을 공개 상태가 양쪽에서 일치한다.
- [ ] 펭귄 마을 대체 효과는 체인 블록을 만들지 않는다.
- [ ] 펭귄 마을 대체로 묘지로 보낸 몬스터 이벤트가 정상 발생한다.
- [ ] 펭귄 마을 대체로 묘지로 보내진 뒤 관련 묘지 회수 유발이 후보로 뜬다.

## 각성의 펭귄 군단

- [ ] ① 반복마다 확인/취소를 묻는다.
- [ ] 확인을 누르면 1회 더 처리한다.
- [ ] 취소를 누르면 반복을 종료한다.
- [ ] 버릴 패가 없으면 자동 종료하고 알린다.
- [ ] 상대 필드에 처리 가능한 몬스터가 없으면 자동 종료하고 알린다.
- [ ] 묘지에 소환 가능한 펭귄 몬스터가 없으면 자동 종료하고 알린다.
- [ ] 내 몬스터 존이 가득 차면 자동 종료하고 알린다.

## 평화의 펭귄

- [ ] ①은 상대 효과 발동에 응답하는 퀵 효과로 뜬다.
- [ ] 상대 필드 몬스터가 없으면 발동할 수 없다.
- [ ] 자신 필드 펭귄 몬스터가 없으면 발동할 수 없다.
- [ ] ②는 묘지에서 제외를 코스트로 지불한다.
- [ ] 덱에 펭귄 마을이 없으면 ②를 발동할 수 없다.
- [ ] 패를 버릴 수 없으면 ②를 발동할 수 없다.
