# 지배자/지배룡 수동 테스트 체크리스트

- [ ] 패 버림을 요구하는 효과는 패가 없으면 발동할 수 없다.
- [ ] `discarded` 이벤트와 `sentToGrave` 이벤트가 구분된다.
- [ ] 지배룡 ②는 버려졌을 때만 후보로 뜬다.
- [ ] 패 이외에서 묘지로 보내졌을 경우 효과는 `sourceZone !== hand`일 때만 후보로 뜬다.
- [ ] 상대 효과 무효는 올바른 태그/타이밍에서만 발동 가능하다.
- [ ] 4종류 조건이 부족하면 사원소 지배자/지배룡 소환 버튼이 뜨지 않는다.
